# SELECT版FTP:
## 项目介绍
```
使用SELECT或SELECTORS模块实现并发简单版FTP
允许多用户并发上传下载文件
```

## 程序介绍
- 作者：tengjuyuan
- 版本：v1.0
- 程序功能：本程序使用了SELECT模块多并发的FTP
```
使用前需要将上传的文件存放在clinet文件夹下面，
需要下载的文件存放在server文件夹里面

上传命令：
    push file_name
下载命令:
    pull file_name

注：在cmd中可以很好的显示进度条，但是不能很好的显示字体颜色，
    在pycharm里面可以很好的显示字体颜色，但是不能正常显示进度条
```
